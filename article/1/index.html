Billie Eilish Fan Page

https://zasz123.github.io/billieEilistFanPage/

미국 아티스트인 Billie Eilish의 팬 페이지입니다.

Billie Eilish의 앨범, 화보, 영상등을 사용하였습니다.


프로젝트 계기 및 목적

해당 프로젝트는 웹 디자인 이해도와 실력을 키우기 위해 진행했습니다.

또, 프로젝트를 시작했을 당시 Billie Eilish의 생일이 얼마 안남아서 스스로 축하해주고자 시작했던 프로젝트이기도 합니다.


사용한 기술 및 툴

  • HTML
  • CSS (BEM)
  • JavaScript
  • Adobe Photoshop

프로젝트

Billie Eilish를 악동, 기존의 룰을 깨는 아티스트라 생각합니다.

하지만 저는 내면의 깊은 이야기를 하고, 진실된 이야기를 하는 아티스트라 생각합니다.


어두운 색상과 정리된 레이아웃을 선택하여 깊고 진실된 내면을 표현했습니다.

또한 Billie Eilish의 개성이 잘 나타나는 아이템(앨범 커버, 화보 이미지 등)들을 사용해서 재미를 더했습니다.


그렇게 Main Image, Music, Video, Store 부분을 나누어서 디자인을 진행했고, 각 파트가 나올 때마다 구현을 함께 진행했습니다.

페이지 특성상 재사용하는 컴포넌트가 많기 때문에 CSS의 BEM 방법론으로 재사용성을 높였습니다.


어려웠던 점


1. 디자인

포토샵을 처음 사용했던 때라 사용법을 익히는 것이 많이 어려웠습니다.

Pinterest에서 랜딩 페이지들을 가져와 수정하고, 새로운 요소를 추가하는 등의 연습으로 포토샵을 익혔습니다.


또, MUSIC 파트에서 앨범 커버들의 위치 선정이 어려웠습니다.

최대한 색조가 비슷한 이미지끼리 붙이고, 여러번의 수정끝에 어색하지 않게 위치선정을 하게 되었습니다.


2. BEM 방법론

어떻게 깔끔한 코드를 유지할 수 있을까 고민하던 중 CSS의 BEM 방법론을 찾게 되었습니다.

BEM은 Block Element Modifier의 앞 글자를 딴 말로, 세 가지 개념으로 CSS 코드를 관리하는 방법론입니다.


처음 익힐 때는 Block과 Element를 어떻게 구분해야 할지 정말 헷갈렸습니다.

또한 사람들마다 사용하는 방법이 조금씩 달라서 학습하는 과정에서 잘 사용하는 방법을 정말 많이 고민했던 것 같습니다.


아쉬웠던 점

1. 아이템(앨범 커버, 화보 등)

포토샵을 통해 수정한 이미지들(앨범 커버, 화보 사진) 등을 배포할 수 없었던 점이 가장 아쉬웠습니다.

나중에 기회가 된다면 유료 이미지들을 구입해서 다시한번 디자인 해보고 싶습니다.


2. MUSIC 파트

해당 파트는 Billie Eilish의 앨컴 커버 이미지들을 Grid 레이아웃을 사용한 파트입니다.

앨범 커버들의 색상이나 분위기들을 파악해서 배치하는데만 신경을 썼습니다.

그러다보니 전체적인 크기는 신경쓰지 못하게 되었습니다...


디자인 툴에서 보면 정말 이쁘지만 정작 웹 페이지에서는 한 화면에 다 볼 수 없었습니다.

디자인 툴에서 보는 사이즈와 실제 웹 페이지에 적용한 사이즈는 많이 다르다는 것을 깨닫게 되었습니다.
